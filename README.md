## Cara Menjalankan Secara Lokal

**Prasyarat:** Node.js v18 atau v20+

1. Instal dependensi:
   ```bash
   npm install
   ```
2. Buat file `.env` dan atur kunci API Anda (lihat `.env.example` sebagai referensi).
3. Jalankan server pengembangan lokal:
   ```bash
   npm run dev
   ```
